Shows most 5 popular post in the Website, in a widget and also displays hit counter in the All Post page.

== Description ==
After activation this plugin will show the hit-counter in All Posts page,also provides a widget which will display
top five most viewed post. It also have a feature will will allow user to check/unchecked if the sticky post will be included in the cont or not.
